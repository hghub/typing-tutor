# Tool Content & Feature Audit

Use this alongside `OPERATIONS_PLAYBOOK.md` when deciding:
- which tools need stronger blog coverage
- which tools deserve a practical feature upgrade
- which clusters can support a related new tool

Update this doc whenever a meaningful audit pass is done.

## Current Focus

Priority right now:
1. make sure every important tool has at least one real, need-based blog
2. keep Pakistan-first tools easier to discover and understand
3. improve high-potential tools with small practical features before adding too many new ones

## Tool Observations

### Pakistan ID & Tax Hub
- Content gap: needs more discoverability than raw demand currently gives it
- Useful next blog angles:
  - filer vs non-filer simple explainer
  - how to understand withholding tax rates before a transaction
- Feature gap:
  - simple “common tax checks” quick links block
  - easier copy/share for decoded CNIC or reference output

### Loan Manager
- Content gap: good tool, but most people do not search for the exact tool name
- Useful next blog angles:
  - how to record family borrowing clearly
  - how to avoid confusion in partial repayments
- Feature gap:
  - reminder dates
  - simple export or WhatsApp reminder copy

### Color Palette Generator
- Content gap: strong developer/design use case, but needs more practical entry pages later
- Useful next blog angles:
  - dark-mode color palette tips
  - how to check text readability on dark backgrounds
- Feature gap:
  - lock selected swatches while regenerating others
  - brand-palette presets

### Symptom Context Tracker
- Content gap: useful health tracker, but needs stronger plain-language entry content
- Useful next blog angles:
  - headache tracking
  - fatigue and weather tracking
- Feature gap:
  - trigger tags like sleep, food, stress
  - exportable summary for doctor visits

### Measurement Tracker
- Content gap: broad tool that needs more scenario-driven discovery
- Useful next blog angles:
  - tracking blood pressure
  - tracking home/project measurements
- Feature gap:
  - multiple saved dashboards
  - goal thresholds and reminders

### Data Transformer
- Content gap: now has a direct blog, but can support more privacy/data-work content later
- Useful next blog angles:
  - de-identifying CSV before sharing
  - preparing data for AI safely
- Feature gap:
  - field-level include/exclude controls
  - saved transform templates

### Config Converter
- Content gap: direct blog exists now, but cluster can expand around DevOps and local setup pain
- Feature gap:
  - side-by-side diff after conversion
  - comments preservation notes when possible

### Text Diff Checker
- Content gap: direct blog exists now
- Feature gap:
  - export diff as markdown or HTML
  - URL/state share for quick reviews

### Student Group Randomizer
- Content gap: direct blog exists now, but classroom cluster can grow
- Useful next blog angles:
  - how to avoid repeating the same pairs
  - project group balancing for mixed-ability classes
- Feature gap:
  - seat-plan mode
  - “avoid previous pairing” memory across sessions

### Markdown Scraper
- Content gap: still weak direct coverage
- Recommended next blog:
  - how to turn a webpage into clean markdown for AI or notes
- Feature gap:
  - paste article HTML and compare output presets
  - keep/remove links toggle

### Currency Converter
- Content gap: still weak direct coverage
- Recommended next blog:
  - how to budget a trip when prices are in AED, SAR, USD, and PKR
- Feature gap:
  - exchange-fee / spread input
  - saved trip budgets

### Packing List
- Content gap: still weak direct coverage
- Recommended next blog:

- Feature gap:
  - trip-type presets
  - passport/visa/doc checklist block

### Warranty Tracker
- Content gap: still weak direct coverage
- Recommended next blog:
  - how to track electronics and appliance warranties before they expire
- Feature gap:
  - calendar export
  - receipt image attachment or note block

### Freelancer Risk Analyzer
- Content gap: still weak direct coverage
